In Visual Studio Code im Backend Ordner eine Konsole öffnen:
	-> alle nötigen Module installieren (siehe Fehlermeldung bei Versuch mit "node server.js")
	-> mit "node server.js" oder "nodemon server.js" (automatische Aktualisierung) den Backend-Server starten
	-> im Browser über localhost:8000 errreichbar 
	-> JSON Objekte jeweils z.B. über "localhost:8000/api/rooms/all" anzeigen


In Visual Studio Code im Frontend Ordner eine Konsole öffnen:
	-> alle nötigen Module installieren (siehe Fehlermeldung bei Versuch mit "node server.js")
	-> mit "node app.js" oder "nodemon app.js" (automatische Aktualisierung) den Frontend-Server starten
	-> im Browser über localhost:3000 alle HTML Seiten erreichbar
